# AI実装レビュー仕様書

最終更新: 2026-02-15  
対象: Next.js + Tailwind で実装された「X拡散型感情コンテンツ（オラクル型）」

## 1. 目的

AI実装を次の3軸で機械的に判定する。

- バズ設計として成立しているか
- UI/UXとして破綻していないか
- 技術構成として保守可能か

## 2. レビュー観点

### 2.1 UX

- 入口1秒理解（何をするサービスか）
- 主導線が1本化されている
- 結果表示までのテンポが速い
- 自分向け感のある文言
- 回遊導線（深掘り/再挑戦）が自然

### 2.2 バズ導線

- 結果0.5秒以内に刺さる一文が見える
- 共有CTAが視認しやすい
- 投稿文テンプレが完成している
- ハッシュタグがテーマ連動で自動付与
- 夜投稿向け余韻文がある

### 2.3 スクショ設計

- 375pxで1画面完結
- タイトル/テーマ/カード名/日付/署名が入る
- 可読性（行間/コントラスト）が十分
- スクショ後に共有導線へ接続できる

### 2.4 Tailwind設計

- ユーティリティ中心で統一されている
- 不要な任意値乱用がない
- 色/余白/角丸の規則性がある
- 画面間でUIパターンを再利用している

### 2.5 Next.js構造

- App Routerの責務分離が適切
- ルーティングが要件準拠（`/` `/draw` `/result` `/deep`）
- Client/Server境界が明確
- API I/Oが後方互換を壊さない

### 2.6 パフォーマンス

- 初回体感速度が遅くない（特に result）
- 不要な再レンダリングを避けている
- API失敗時フォールバックがある

### 2.7 将来拡張性

- 計測イベントに拡張余地がある
- 文生成/共有文/計測が分離されている
- 将来の課金・ログイン追加で衝突しない

### 2.8 スマホ最適

- safe-area対応
- タップ領域44px以上
- 親指到達しやすいCTA配置
- 長文時に崩れない

## 3. OK/NG判定基準（厳しめ）

### 3.1 即NG（1つでも該当で不合格）

- テーマ未選択で結果遷移できる
- 結果画面でタイトル先表示がない
- スクショ1画面完結が崩れる
- 共有テンプレが3種未満 or 壊れている
- 深掘り上限が効かない
- 必須イベントが送信されない
- API失敗で結果表示不能

### 3.2 条件付きOK

- 文言トーンの軽微な不統一
- Tailwindクラスの冗長
- 小さな余白/視認性の改善余地

### 3.3 完全OK

- 即NG項目ゼロ
- 必須要件100%達成

## 4. レビュー時チェックリスト（YES/NO）

### 4.1 体験要件

- [ ] YES/NO: 入口で1秒以内に意図が理解できる
- [ ] YES/NO: テーマ未選択時にCTAが無効
- [ ] YES/NO: `/draw` が0.6〜0.9秒で完了し `/result` へ自動遷移
- [ ] YES/NO: `/result` でタイトルが先に表示される
- [ ] YES/NO: 本文は後段表示（視線競合がない）
- [ ] YES/NO: `/deep` は2回上限が機能する

### 4.2 バズ・共有

- [ ] YES/NO: 共有CTAが結果画面の主導線
- [ ] YES/NO: 共有テンプレ3種（short/emotional/night）
- [ ] YES/NO: テーマ連動ハッシュタグ自動付与
- [ ] YES/NO: 夜投稿向け余韻文がある
- [ ] YES/NO: 投稿文が手直しなしで使える

### 4.3 スクショ

- [ ] YES/NO: 375pxで1画面完結
- [ ] YES/NO: 必須情報（タイトル/テーマ/カード名/日付/署名）を含む
- [ ] YES/NO: 文字可読性が十分
- [ ] YES/NO: スクショ後の共有導線がある

### 4.4 技術品質

- [ ] YES/NO: App Router構造が破綻していない
- [ ] YES/NO: API変更が後方互換を維持
- [ ] YES/NO: 必須イベント送信が動作する  
  `theme_selected` `result_first_paint` `share_clicked` `deep_dive_opened`
- [ ] YES/NO: エラー時フォールバックあり
- [ ] YES/NO: 状態管理の責務が明確

### 4.5 スマホ

- [ ] YES/NO: safe-area対応
- [ ] YES/NO: タップ領域44px以上
- [ ] YES/NO: 主要CTAが親指到達範囲
- [ ] YES/NO: 長文でレイアウト崩れなし
